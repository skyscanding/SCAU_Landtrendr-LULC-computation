# SCAU_Landtrendr-LULC-computation
SVM-based LULC classification for Dabaoshan mine area via Google Earth Engine. Supports Landsat 7/8/9 and Sentinel-2 with RBF-kernel SVM, multi-year compositing, and best-per-year accuracy gating. Runnable as GEE Code Editor JavaScript or local Python CLI with GeoTIFF output.
